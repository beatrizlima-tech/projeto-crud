# CRUD de Usuários

Projeto desenvolvido para praticar conceitos de desenvolvimento backend utilizando Java e Spring Boot.

## Sobre o Projeto

Esta aplicação implementa um sistema CRUD (Create, Read, Update e Delete) para gerenciamento de usuários através de uma API REST.

## Tecnologias Utilizadas

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven

## Funcionalidades

* Criar usuário
* Listar usuários
* Buscar usuário por ID
* Atualizar usuário
* Excluir usuário

## Estrutura do Projeto

* Controller
* Service
* Repository
* Model

## Como Executar

### Pré-requisitos

* Java 17 ou superior
* Maven
* MySQL

### Clone o projeto

```bash
git clone https://github.com/beatrizlima-tech/projeto-crud.git
```

### Execute a aplicação

```bash
mvn spring-boot:run
```

Durante o desenvolvimento deste projeto foram praticados:

* Arquitetura em camadas
